# 文件搬迁助手
你的某个磁盘分区用着用着发现文件占用空间太多，磁盘告警。你想做点什么却被以下场景所阻碍：

- 文件不能被删除，因为需要使用
- 系统盘不能直接被扩展，或分区不在同一个物理盘上无法扩展
- 想移动整个目录，又怕读取这个目录的软件出错
- 文件被占用不知如何处理
- 当你听说有一招从天而降的撑法叫软链接，却发现命令晦涩又麻烦

如果你被以上问题困扰，那么此工具就是为你准备的。

## 使用方式
``` sh
$0 src dist
```
